Basic Chatbot

This is a simple Python chatbot that interacts with users through text-based conversations. The chatbot responds to predefined questions and greetings and continues the conversation until the user types "bye".

Features:

- Greeting responses
- Answers common questions
- Continuous conversation loop
- Exit command

How to Run:

1. Save the file as "chatbot.py".
2. Open it in Python IDLE.
3. Press F5 or click Run → Run Module.
4. Start chatting with the bot.
5. Type "bye" to exit the chatbot.
